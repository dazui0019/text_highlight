# Text Highlight Chrome Extension

一个最小可用的 Chrome 扩展，用来高亮网页中的指定字符串。

## 使用方法

1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `d:\File\AI\Vibe\text_highlight`

## 功能

- 在弹窗中按行输入高亮规则
- 点击“开始高亮”后，对当前页面进行匹配高亮
- 点击“清除高亮”后，移除当前页面中的高亮
- 自动保存上次输入的字符串，刷新页面后仍可继续使用
- 支持普通文本和 `/正则/flags` 混合输入

## 说明

- 普通文本默认使用不区分大小写的匹配
- 正则写法示例：`/OpenAI\s+API/i`
- 在插件代码有更新后，需要到 `chrome://extensions/` 点一次“重新加载”
- 如果当前标签页是本地 `file://` 页面，还需要在扩展详情里开启“允许访问文件网址”
- `chrome://`、扩展商店等受限页面无法注入内容脚本
